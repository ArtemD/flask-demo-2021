# AlcoDB

Simple project powered by Flask/Python/SQLAlchemy for displaying alchohol licensed locations in Finland. Based on avoindata.fi data.


## Installation

Make sure you have Python 3.X installed as well as pipenv.

```bash
  pipenv install
  pipenv shell
  python app.py
```
